  • (3:40) "Browser window" is the more correct term; more on viewports here: https://www.w3schools.com/css/css_rwd_viewport.asp
  • (5:05) All of these device types were deprecated/discontinued with the switch from CSS2 to CSS3; the four discussed here (All, Print, Screen, Speech) are part of the current CSS3 spec.
  • (6:00) The comma can be used to combine several media types, such as @media screen, print. The 'only' operator is not discussed because it has no effect on modern browsers: https://www.w3schools.com/cssref/css3_pr_mediaquery.asp
  • (8:45) Notice that these viewport widths don't overlap; overlapping viewport widths can cause unexpected results! Also note that I could have used the min-width statement only and omitted the max-width; this would have yielded the same result, but for my own sanity, I always specify max-width for my own sanity.
  • (11:25) Meant to say pseudo-element, not pseudo-class. More on those here: https://developer.mozilla.org/en-US/docs/Learn/CSS/Introduction_to_CSS/Pseudo-classes_and_pseudo-elements
